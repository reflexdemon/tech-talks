# Spec: OpenSpec Workflow

## ADDED Requirements

### Requirement: Detail the OpenSpec Artifact Chain
The presentation must walk through each level of the OpenSpec workflow: Proposal, Design, Specs, and Tasks.

#### Scenario: Visualizing the sequence
- **WHEN** Viewing the workflow overview slide
- **THEN** A Mermaid diagram must show the flow from intent (Proposal) to execution (Tasks).

#### Scenario: Explaining "Artifact-Driven implementation"
- **WHEN** Discussing the realization phase
- **THEN** The presentation should explain how AI agents use specs/tasks to generate code, ensuring zero "hallucination" of requirements.

# Capability: Java Integration

## Purpose
Demonstrating how Spec-Driven Development integrates with Java projects.

## Requirements

### Requirement: Reducing Java Boilerplate
The presentation must demonstrate how OpenSpec reduces manual effort in Java projects.

#### Scenario: Demonstrating Record generation
- **WHEN** Discussing Java 14+ specific features
- **THEN** Show how a spec for a data object is converted to a Java Record without manual coding.

#### Scenario: Spec-driven testing
- **WHEN** Explaining the "Quality" aspect of SDD for Java
- **THEN** Highlight how Gherkin-style scenarios in specs automatically drive the creation of JUnit/AssertJ tests.
